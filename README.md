# JuegoAhorcado
Proyecto de juego del ahorcado hecho en el lenguaje de GO
Al iniciar el programa se nos desplegara un menu donde podemos elegir dos opciones,jugar o salir del programa,luego otro menu nos mostrara las tres opciones que serian los temas para jugar para despues  poder comenzar a jugar,cada error que tengamos hara que nuestro monito vaya formandose,o si acertamos los espacios vayan llenandose,el programa nos detectara si ponemos la misma letra mas d una ves o si ponemos mas de 1 caracter a la ves desplegando asi un  mensaje segun lo q se haya puesto ,si completamos correctamente la palabra nos aparecera las felicitaciones ademas de la opcion de regresar al menu,si nos equivocamos las suficientes veces oaparecera el monito colgado ademas de un mensaje que nos diga que perdimos
